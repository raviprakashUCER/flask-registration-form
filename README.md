# Flask Registration Form Project

## 🚀 About the Project
This is a simple Flask web application that allows users to register using a form.  
It demonstrates basic backend development using Flask, form handling, and routing.

---

## ✨ Features
- User Registration Form
- Form Validation
- HTML Template Rendering
- Flask Routing System
- Clean UI using HTML

---

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- Jinja2 Templates
